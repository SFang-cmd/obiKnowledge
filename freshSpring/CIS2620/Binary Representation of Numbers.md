$101 = 1 \times 2^{2}+ 0 \times 2^{1} + 1 \times 2^{0} = 5$
$1101 = 1 \times 2^{3} + 1 \times 2^{2}+ 0 \times 2^{1} + 1 \times 2^{0} = 13$

Largest binary representation using $n$ bits is
$111...1 = 1*2^{n-1} + 1*2^{n-2}+...+1*2^{0}= 2^{n-1}+2^{n-2}+...+1=2^{n}-1$
